class chartbook.plotting.backends._hookspecs.PlottingHookSpec[source]#

Hook specifications for chartbook.plotting backends.

Third-party packages can implement these hooks to provide custom plotting backends. Register via entry points:

[project.entry-points.chartbook_plotting]
my_backend = "mypackage.backend:MyBackend"

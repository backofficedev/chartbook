chartbook.manifest.resolve_platform_path(path_input: Union[str, dict]) → pathlib.Path[source]#

Resolves a path that can be either a direct path string or a dictionary of platform-specific paths.

Parameters:

path_input (Union[str, dict]) – Either a string representing a direct path, or a dictionary containing platform-specific paths with ‘Windows’ and/or ‘Unix’ keys.

Returns:

The resolved path appropriate for the current platform.

Return type:

Path

Raises:

ValueError – If using a dict input and no valid path is found for the current platform.

Examples::

resolve_platform_path('/path/to/dir')
# Returns: PosixPath('/path/to/dir')

resolve_platform_path({'Windows': 'C:/data', 'Unix': '/home/data'})
# Returns: WindowsPath('C:/data') on Windows, PosixPath('/home/data') on Unix/macOS

chartbook.manifest.validate_doc_fields(manifest: dict, path_key: str, str_key: str, object_type: str, object_id: str) → tuple[str, str][source]#

Validate that exactly one of path or str doc fields is provided.

Parameters:

  • manifest – The manifest dictionary for the object (dataframe or chart).

  • path_key – The key for the path-based documentation field.

  • str_key – The key for the inline string documentation field.

  • object_type – The type of object (‘dataframe’ or ‘chart’) for error messages.

  • object_id – The ID of the object for error messages.

Returns:

A tuple of (mode, value) where mode is ‘path’ or ‘str’.

Raises:

ValueError – If both or neither fields are provided.

Examples::

validate_doc_fields(
    {'dataframe_docs_path': './docs/df.md'},
    'dataframe_docs_path', 'dataframe_docs_str',
    'dataframe', 'my_df'
)
# Returns: ('path', './docs/df.md')

validate_doc_fields(
    {'dataframe_docs_str': 'Inline docs here'},
    'dataframe_docs_path', 'dataframe_docs_str',
    'dataframe', 'my_df'
)
# Returns: ('str', 'Inline docs here')
chartbook.manifest.validate_os_compatibility(value: Union[str, list]) → Union[str, list][source]#

Validate that os_compatibility is either a string or a list of strings.

Parameters:

value (Union[str, list]) – The os_compatibility value to validate.

Returns:

The validated value (unchanged).

Return type:

Union[str, list]

Raises:

TypeError – If the value is neither a string nor a list of strings.

Examples::

validate_os_compatibility("Windows/Linux/macOS")
# Returns: "Windows/Linux/macOS"

validate_os_compatibility(["Windows", "Linux", "macOS"])
# Returns: ["Windows", "Linux", "macOS"]

validate_os_compatibility(123)
# Raises: TypeError
